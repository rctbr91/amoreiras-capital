## ⚙️ Como Ativar o Website

### 1️⃣ Habilitar GitHub Pages

1. Vá para **Settings** → **Pages**
2. Em "Build and deployment", selecione:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (ou `master`)
3. Clique em **Save**
4. Aguarde 2-3 minutos para o site ativar

### 2️⃣ Adicionar Links de Documentos do OneDrive

1. Abra o arquivo `index.html`
2. Clique no ✏️ botão de editar
3. Procure por `https://YOUR_ONEDRIVE_LINK_HERE`
4. Substitua cada um pelos links compartilhados do seu OneDrive:
   - Código de Ética (linha ~26)
   - Manual de Controles Internos (linha ~31)
   - Política de Investimentos Pessoais (linha ~36)
   - Política de Rateio e Divisão de Ordens (linha ~41)
   - Política de Voto (linha ~46)
   - Política de Gestão de Risco (linha ~51)
   - Formulário de Referência (linha ~56)
5. Clique em **Commit changes**

### 3️⃣ Configurar Domínio Personalizado

#### No seu Registrador de Domínio:

1. Acesse as configurações de DNS do seu domínio
2. Adicione um registro **CNAME**:
   - **Nome:** www
   - **Valor:** rctbr91.github.io
   - **TTL:** 3600 (ou deixe padrão)

3. Aguarde 24-48 horas para propagação DNS

#### No GitHub:

1. Vá para **Settings** → **Pages**
2. Em "Custom domain", adicione: `www.amoreirascapital.com.br`
3. Clique em **Save**
4. Marque "Enforce HTTPS" ✓
